从零开始，再次配置orangepi

نُشر في 2017-11-12 عُدل في 2017-11-13 في Tech

回家拿衣服的时候，发现家里有一块旧的orangepi，虽然是个配置很垃圾的小板子，但是还是可以有很多功能的，而且现在我已经会摆弄一些8266的东西，这个orangepi似乎可以应付一阵子了。

刚刷完armbian，在apt-get update的时候报错：

E: Could not get lock /var/cache/
搜了一下，解决方法如下：
sudo rm -rf /var/cache/apt/archives/lock
这样就可以了。

samba

按流程走一套update和upgra之后，安装samba。

apt-get install libcups2 samba samba-common cups
apt-get install samba
安装完成后编辑samba配置，加入root文件夹。
sudo nano /etc/samba/smb.conf

wins support = yes # !打开对Win支持
[root]
    comment = Admin Config Share
    path = /
    valid users = root
    force user = root
    force group = root
    admin users = root   
    writeable = Yes

sudo smbpasswd -a root
sudo systemctl restart smbd.service
大工搞成~！
