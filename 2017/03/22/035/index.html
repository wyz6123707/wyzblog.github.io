写给自己的orangepi启动指南

نُشر في 2017-03-22 في Tech

我的这台orangepi型号是orange Pi zero，搭载的os是官网下载的ubuntu server。电脑的操作系统是mac os。

烧录系统

烧录系统应该就不用讲了，用Etcher太白痴了没啥好说的。

ssh联机

一开始肯定要用网线连接，并通电。
这个os是默认打开ssh服务的，不知道IP地址，所以要打开路由器的管理页面，进入dhcp服务器，查看orangepi的IP地址。比如我这里是192.168.2.2，就在终端里输入

ssh orangepi@192.168.2.2 -p 22

然后输入默认密码 “orangepi”，就可以连接了。

root

设置root账户

su //进入root账户
这时pi会先让你输入orangepi的密码
然后再设置root账户的密码

sudo passwd root
然后再给root账户设一下密码

wifi

很简单，用一句指令就行了。

sudo nmtui-connect your_ssid //用你的路由的ssid吧your_ssid替换了就行了

然后就会让你输入密码，然后就连上了。

扩容

一开始我也被这个问题折腾了很久，其实也是非常简单一行命令。而且在第一次登陆orangepi的时候，pi自己会告诉你需要用这个指令扩容一下。

sudo fs_resize

就是这么简单~
